# Install in Claude Code

1. Download the right pack zip from the latest GitHub release.
2. Unzip it locally.
3. Copy every folder under `skills/` into one of these locations:
   - global: `~/.claude/skills/<skill-name>/`
   - project: `.claude/skills/<skill-name>/`
4. Connect the FirstTouch MCP at `https://mcp.firsttouch.ai`.
5. Start Claude Code and paste the activation prompt from the repository README.
