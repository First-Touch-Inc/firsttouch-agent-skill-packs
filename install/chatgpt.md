# Use with ChatGPT

ChatGPT can connect to the FirstTouch MCP, but it does not currently install Agent Skills folders the same way Claude does.

Recommended setup:

1. Connect the FirstTouch MCP at `https://mcp.firsttouch.ai` in ChatGPT connectors or MCP settings.
2. If using a Custom GPT or project knowledge, upload the selected pack README and skill files as knowledge.
3. Paste the activation prompt from the repository README.

Do not rely on the prompt alone to install files. ChatGPT needs MCP access and, when available, uploaded pack knowledge.
